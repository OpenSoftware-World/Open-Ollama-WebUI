## 🚀 How to Use?

1. Download and extract the OSW-Ollama-WebUI [from here](https://github.com/OpenSoftware-World/Open-Ollama-WebUI/releases).  
2. Move the `Open Ollama WebUI` folder to your project directory.  
3. Run the file named webui.py and enter the address specified in the command line into your browser.
